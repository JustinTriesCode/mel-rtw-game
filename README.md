## Melly's Minalmist Minigames

This project is a collection of small and simple games designed to help with screen reintegration.

Game 1: Alphabet Soup

- How to play: 
  - Easy Mode (Letter Soup): Type the letter shown on the screen. 
  - Hard Mode (Alphabet Soup): Type the letter that matches the colour of the orb on screen.

- Controls: 
  - Keyboard keys.
  - ESC to return to the Main Menu at any time.


Game 2: Equationista

- How to Play:
  - Look at the four values/equations around the center image.
  - Press the Arrow Key or WASD Key corresponding to the side with the lowest number/value.
  - Easy Mode (Numberista): Press the direction of the lowest number.
  - Hard Mode (Equationista): Press the direction of the equation that equals the lowest value.
  - Scoring: You get 10 points for a correct answer on your first try. If you miss, you can try again for reduced points. Bonus points can be earned for longer streaks.
  - The Goal: Keep your streak alive! If the numbers all get too high, the board will reset to give you a fresh start.

- Controls
  - Up/Down/Left/Right or W/A/S/D to play.
  - ESC to return to the Main Menu at any time.
